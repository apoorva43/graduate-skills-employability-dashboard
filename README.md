# Graduate Skills Employability Dashboard

Individual assignment re-implementation of the [group project](https://github.com/UBC-MDS/DSCI-532_2026_12_GradSkills) in Shiny for R.

## Try it yourself

Stable app: add-link

## Setup & Installation

### 1. Clone the repo
```
git clone https://github.com/apoorva43/graduate-skills-employability-dashboard
cd graduate-skills-employability-dashboard
```

### 2. Install R packages
Open RStudio and run:
```
install.packages(c("shiny", "bslib", "dplyr", "ggplot2", "readr", "scales"))
```

### 3. Run the app
```
shiny::runApp("app.R")
```
